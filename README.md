## 简介
获取文件的crc32值

## java
```
javac crc.java
java crc filename
```
### Example:
```shell
java crc vezel/classes.dex

vezel/classes.dex
HEX:46E26557
DEC:1189242199
```

## Python3
```
python crc.py filename
```
## Example:
```shell
python crc.py vezel/classes.dex

Hex:    46e26557
Dec:    1189242199
```
